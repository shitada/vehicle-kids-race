# ✈️ そらの おさんぽフライト — 仕様書

## 1. 概要

- **ゲーム名**: そらの おさんぽフライト（仮）
- **ジャンル**: 横スクロール フライトゲーム
- **対象年齢**: 4〜8歳
- **プラットフォーム**: Webブラウザ（PC / iPad）※クライアントサイド完結
- **技術スタック**: PixiJS（v8）+ TypeScript + Vite
- **UI描画方式**: DOM オーバーレイ（タイトル・HUD・リザルト等のUI）+ PixiJS Canvas（ゲーム描画）
- **視点**: 横スクロール（サイドビュー）
- **デザイン参考**: [のりものリズムレース](https://shitada.github.io/vehicle-rhythm-race/) — ポップ・カラフル・ひらがなUI・絵文字活用
- **UI方針**: テキスト・絵文字・ボタン等のUIはDOM（HTML/CSS）で描画し、常にクリスプな表示を実現。ゲーム本体（飛行機・背景・エフェクト）はPixiJS Canvasで描画

## 2. ゲームコンセプト

- プレイヤーは **飛行機** を操縦して、空のコースをゴールまで飛ぶ
- 敵・障害物は **なし** — 純粋に飛ぶ気持ちよさを楽しむ
- コースは **一方通行**（スタート → ゴール）
- クリアまでの目安時間: **約1分〜1分15秒**
- ブーストを使ってスピード感を味わえる

## 3. ゲームの流れ

```
タイトル画面 → レース（カウントダウン → フライト → ゴール） → リザルト画面
```

1. **タイトル画面**: 「✈️ あそぶ」ボタンを押してスタート
2. **カウントダウン**: 3, 2, 1, GO!
3. **フライト**: 横スクロールで空を飛ぶ（約60〜75秒）
4. **ゴール演出**: ゴールゲート通過 → 星キラキラエフェクト & ファンファーレ
5. **リザルト画面**: タイム表示、「🔄 もういちど」「🏠 タイトルへ」

## 4. 操作方法

### 4.1 PC（キーボード）

| 入力 | キー | 動作 |
|------|------|------|
| 上に移動 | ↑ | 飛行機が上へ移動 |
| 下に移動 | ↓ | 飛行機が下へ移動 |
| 前に加速 | → | 飛行機が前方へ少し加速 |
| 減速 | ← | 飛行機が少し減速 |
| ブースト | Space | 一時的に大幅スピードアップ（クールダウンあり） |
| バレルロール | Shift | 飛行機がくるっと1回転（キラキラエフェクト） |

### 4.2 iPad / タブレット（タッチ操作）

画面上にDOM要素のバーチャルコントローラーを常時表示:

- **左側: 上下パッド**
  - 縦長カプセル型ベース（72×152px）に上下ボタンのみ配置
  - 上下で飛行機を移動
  - 後退キーは廃止（不要なため）
  - 押している間はボタンが明るくなり縮小するフィードバック
- **右側: アクションボタン（3つ）**
  - ▶ **前進ボタン**（64px・水色）— 🔥ブーストの左隣に配置。押している間加速
  - 🔥 **ブーストボタン**（68px・オレンジ）— タップでブースト発動
  - 🌀 **バレルロールボタン**（56px・紫）— ブーストの下に配置。タップで回転
  - クールダウン中はブーストゲージがグレー表示
- **ボタンサイズ**: 子供の指でも押しやすい大きめサイズ（56〜68px）
- **配置**: 全ボタン絶対配置で画面下部に固定（画面外にはみ出さない設計）
- **透明度**: 半透明（opacity: 0.6）、押下時に明るくなるフィードバック

## 5. ゲーム要素

### 5.1 飛行機（プレイヤー）
- 乗り物は **飛行機のみ**（1種類）
- かわいい丸みのあるデザイン（デフォルメ調）
- 飛行時にプロペラ回転 or ジェット噴射のアニメーション
- ブースト時にエフェクト（炎・光の線）
- ブースト時にソニックブーム演出（衝撃波リングが広がる + 「ドォン！」という低音SE）
- バレルロール（Shift / タッチボタン）でくるっと1回転。回転中キラキラエフェクト
- プレイ回数・累計星数でレベルアップ。飛行機の見た目が少しずつ変化:
  - Lv.1: ノーマル
  - Lv.2: 翼にステッカー追加
  - Lv.3: 翼のデザイン変化（カッコいいライン）
  - Lv.4: ジェットエンジンが光る
  - Lv.5: ゴールドカラー（最終形態）
- レベルはlocalStorageに保存

### 5.2 コース
- **一方通行型**: 左から右へ進むスクロール
- コース全長: 約60〜75秒分の距離
- 上下の移動範囲に制限あり（画面外に出られない）
- **4ゾーン構成**（背景テーマが切り替わり飽きさせない）:
  - ゾーン1 🏙️ **まちの うえ**: ビル・おうちの上を飛ぶ朝の空
  - ゾーン2 🌊 **うみの うえ**: 青い海・イルカがジャンプする演出
  - ゾーン3 🏔️ **やまごえ**: 夕焼け空・山の間を縫って飛ぶ
  - ゾーン4 🌙 **よぞら → ゴール**: 星空＆花火の中をゴールへ
- ゾーン切り替え時に短い演出テロップ（例:「うみだ！」）

### 5.3 💨 かぜのトンネル（スピードリング）
- コース上にたまに出現する **光のリング**（3〜5箇所）
- くぐると一瞬加速 +「シュゴォ！」の風SE
- リング通過時にカメラが軽く振動（画面シェイク）
- 障害物ではなく **通るとご褒美** の仕組み（避ける必要なし）

### 5.3 ⭐ コイン（ほし）収集
- コース上に **星型コイン** を配置（合計30〜50個）
- 飛行機が触れると「キラーン♪」の効果音とともに取得
- 取得時に星が弾けるキラキラエフェクト
- 画面左上に取得数を表示（例: ⭐ 12 / 40）
- リザルト画面で収集率を表示
- コインの配置パターン:
  - 直線に並ぶ列
  - アーチ状（上下に動く誘導）
  - ボーナスゾーン（密集エリア）

### 5.5 スピードメーター
- 画面右下に常時表示
- 現在速度をアナログ風メーターで表示
- ブースト時にメーターが振り切れる演出

### 5.6 ブースト
- Spaceキー or タッチボタンで発動
- 効果: 約2〜3秒間、速度が1.5〜2倍に
- クールダウン: 約5秒（ゲージで残り時間を表示）
- ブースト可能時にアイコンが光るUI

### 5.7 進捗表示
- 画面上部にプログレスバー（スタート〜ゴールの進行度）
- 飛行機アイコンが現在地を示す
- ゾーン区切りもバー上に表示

### 5.8 ✨ 飛行機の軌跡エフェクト（トレイル）
- 飛行機の後ろにカラフルな煙のトレイルが常時表示
- 通常時: 白〜水色の軽い煙
- ブースト時: 虹色 or 炎色のド派手なトレイルに変化
- トレイルは飛行機の動きに合わせてなびく（上下移動で曲線を描く）

### 5.9 🏅 タイム評価 & 称号
- リザルト画面でタイムに応じた評価を表示:
  - ⭐⭐⭐ + 「🏆 スカイマスター！」（目標タイム以内）
  - ⭐⭐ + 「✈️ はやい！すごい！」（そこそこ速い）
  - ⭐ + 「☁️ のんびりフライト♪」（ゆっくりでもポジティブ）
- どの結果でも褒める文言にする（子供が落ち込まない設計）

### 5.10 📸 ゴール時のスクリーンショット演出
- ゴール通過の瞬間に画面が一瞬フラッシュ（白）
- 「パシャ！」のカメラシャッター音
- ゴール時の飛行機ポーズがフレーム付きで表示される
- リザルト画面にそのスナップショットが「きねんしゃしん」として表示

### 5.11 🔁 ベストタイム記録
- localStorage にベストタイムを自動保存
- タイトル画面に「🏆 ベストタイム: ○○びょう」を表示
- ベスト更新時にリザルト画面で「🎉 きろく こうしん！」の特別演出
- 星の最大収集数も記録

## 6. 画面構成

### 6.1 タイトル画面
- ゲームタイトル（大きなひらがな + 飛行機アイコン）
- 「✈️ あそぶ」ボタン（大きくタップしやすい）
- 🏆 ベストタイム表示
- ⭐ ほし さいだい取得数
- 🔊 おと ON/OFF ボタン

### 6.2 レース画面
- **背景**: パララックス（多層スクロール）で奥行き感
  - 最奥: 空・雲（ゆっくり）
  - 中間: 遠景の山・建物（中速）
  - 手前: 地面・木（速い）
- **飛行機**: 画面左寄り 1/3 あたりに固定表示
- **UI要素**:
  - 上部: プログレスバー（コース進行度 + ゾーン表示）
  - 左上: ⭐ コイン取得数
  - 右下: スピードメーター
  - 左下: ブーストゲージ
  - 中央上: タイマー（経過時間）

### 6.3 リザルト画面
- 🏆 「ゴール！」の大きなテキスト
- 📸 きねんしゃしん（ゴール時スナップショット）
- ⏱️ クリアタイム（ベスト更新時は「🎉 きろく こうしん！」）
- ⭐ 評価（星1〜3 + 称号テキスト）
- ⭐ コイン収集率（例: 35 / 40）
- 「🔄 もういちど」ボタン
- 「🏠 タイトルへ」ボタン

## 7. 演出・デザイン方針

### デザインスタイル（参考: のりものリズムレース）
- **テキスト**: すべて **ひらがな**（4〜5歳が読める）
- **配色**: パステルカラー + ビビッドなアクセント
- **フォント**: 丸ゴシック系（太め・読みやすい）
- **ボタン**: 大きく・角丸・押した感のあるアニメーション
- **絵文字**: UIに絵文字を積極活用（✈️🏆⭐🔊🔄🏠）
- **全体の雰囲気**: 安心感のある、やさしいトーン

### アニメーション・エフェクト
- スタート時: カウントダウン（数字が大きく表示 → 消える）
- 飛行中: 雲がふわふわ流れる、鳥がたまに横切る
- コイン取得: キラキラパーティクル + 「キラーン♪」音
- ゾーン切替: テロップ演出（例:「🌊 うみだ！」がふわっと表示）
- ブースト: 画面が少しズームアウト + スピードライン + 虹トレイル
- ゴール時: 画面フラッシュ →「パシャ！」→ 星エフェクト
- ゴール: 金色の星パーティクル（Graphics描画・軽量20個） + ファンファーレ音

### サウンド
- BGM: 明るく軽快な曲（ループ）
- **BGMとスピード連動**: スピードが上がるとBGMのテンポが少し加速。ブースト時は盛り上がりパートに切り替わる
- 効果音: プロペラ / ジェット音、ブースト発動音（ソニックブーム）、バレルロール音、コイン取得音、風のトンネル通過音、カメラシャッター音、ゴールファンファーレ、レベルアップ音
- 🔊 ボタンで ON/OFF 切り替え
